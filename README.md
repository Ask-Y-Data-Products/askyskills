# AskySkills

A collection of Claude Code skills for the Asky data analytics platform.

## Skills

### asky-e2e
Automate Asky platform UI testing using Claude Chrome extension (MCP browser tools). Covers login, project management, file upload, chat interaction, explorer navigation, table data reading, and assertions.

## Usage

These skills are designed to be used with [Claude Code](https://claude.com/claude-code). Copy the desired skill into your `~/.claude/skills/` directory to use it.
